Curated Projects

currency converter

The Currency Converter is a dynamic web tool designed to facilitate real-time currency conversion. It allows users to select currencies, input an amount, and instantly view accurate conversions based on the latest exchange rates.

  • User-Friendly Interface: Clean and intuitive design tailored to site aesthetics.
  • Real-Time Exchange Rates: Fetches data from FloatRates for accurate conversions.
  • Automatic Defaults: Preloaded with default currency pairs and amounts for quick use.
  • Error Handling: Displays clear error messages for invalid inputs (e.g., negative values).
  • Timestamped Data: Presents data in UK BST format for transparency.
fuel forecaster

The Fuel Forecaster is an interactive visualization tool for analyzing historical heating fuel costs in the UK and predicting future trends using real-world data.

  • Comprehensive Visualization: Displays price trends for solid fuels, gas, electricity, and liquid fuels.
  • Historical Data Scrolling: Navigate between January 1996 and August 2022 to examine price changes.
  • Forecasting Capability: Predicts fuel costs for the next 12 months using a straight-line method.
  • Filter Options: Toggle specific fuel types on the chart for customized views.
  • Two Modes: Historical and Forecast modes for exploring past and future data.
